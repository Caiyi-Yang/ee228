**********************************
运行环境：Anaconda3+py3.6+keras-tensorflow
文件说明：
1. train: 训练集及其标签，请将该目录下文件换成所有训练集
2. test：待预测测试集，请将该目录下文件换成所有测试集
3. lib:
(1) models: 包含训练模型文件
(2) data_aug: 包含一些数据增强的操作
(3) utills: 用于画图，数据测试等操作
4. train.py：最终训练脚本
5. test.py： 预测脚本，输出submission.csv供助教老师验证使用
6. model.h5：最终训练模型文件
7. sampleSubmission.csv：用于candidate读取操作
8. train_val.csv: 训练样本的标注文件
